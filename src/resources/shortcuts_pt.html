Atalhos
  • Avançar um Quadro: Seta Direita (→)
  • Voltar um Quadro: Seta Esquerda (←)
  • Abrir o Diálogo de Quadros: Alt
  • Duplicar Quadro: Tecla D
  • Inserir Quadro para a Frente: Tecla Inserir ou Shift + I
  • Copiar Quadro: Ctrl + C
  • Colar Quadro Adiante: Ctrl + V
  • Desfazer: Ctrl + Z
  • Refazer: Ctrl + Y
  • Remover Quadro: Tecla X
  • Reproduzir/Parar Animação: Barra de Espaço ou Tecla Enter
  • Sair do Aplicativo: Esc ou Tecla Q
