Shortcuts
  • Forward One Frame: Right Arrow (→)
  • Back One Frame: Left Arrow (←)
  • Open Frames Dialog: Alt
  • Clone Frame: D Key
  • Insert Frame Forward: Insert Key or Shift + I
  • Copy Frame: Ctrl + C
  • Paste Frame Forward: Ctrl + V
  • Undo: Ctrl + Z
  • Redo: Ctrl + Y
  • Remove Frame: X Key
  • Play/Stop Animation: Spacebar or Enter Key
  • Exit App: Esc or Q Key
