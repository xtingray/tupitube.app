Atajos
  • Avanzar un Marco: Flecha Derecha (→)
  • Retroceder un Marco: Flecha Izquierda (←)
  • Abrir Diálogo de Marcos: Alt
  • Clonar Frame: Tecla D
  • Insertar Frame Adelante: Tecla Insert ó Shift + I
  • Copiar Marco: Ctrl + C
  • Pegar Marco Adelante: Ctrl + V
  • Deshacer: Ctrl + Z
  • Rehacer: Ctrl + Y
  • Eliminar Marco: Tecla X
  • Reproducir/Detener Animación: Barra Espaciadora ó Tecla Enter
  • Cerrar Aplicacion:: Esc ó Tecla Q
